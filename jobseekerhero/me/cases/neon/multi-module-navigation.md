# Multi-module Navigation Modernization - Neon

## Context
Modernization of multi-module navigation by unifying deep links and push notifications with Dagger/Hilt multibindings.

## Challenge/Problem
Navigation across multiple modules was complex and not well-integrated. Deep links and push notifications needed better coordination, and there was a need for automated testing and code independence.

## Your Role
Android Developer - Contributing to the development of essential features and improvements for Neon's Android banking apps, impacting more than 10 million clients.

## Actions Taken
- Modernized multi-module navigation
- Unified deep links
- Integrated push notifications with multibindings using Dagger/Hilt
- Enabled automated testing
- Improved code independence

## Results
- **Improved app navigation**
- **Enabled automated testing** of navigation flows
- **Enhanced code independence** between modules
- Better integration of deep links and push notifications

## Technologies Used
- Dagger/Hilt
- Deep Links
- Push Notifications
- Multibindings
- Kotlin
- Android SDK
