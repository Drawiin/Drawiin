# Gradle Tools and Convention Plugins - Neon

## Context
Development of Gradle tools, including convention plugins, to reduce configuration fatigue and ensure consistent configurations across the project.

## Challenge/Problem
Configuration fatigue was high, and there was inconsistency in configurations across internal libraries, resource modules, and micro apps. This was slowing down development and causing maintenance issues.

## Your Role
Android Developer - Contributing to the development of essential features and improvements for Neon's Android banking apps, impacting more than 10 million clients.

## Actions Taken
- Developed Gradle tools
- Created convention plugins
- Standardized configurations across the project
- Automated configuration setup

## Results
- **80% reduction in configuration fatigue**
- **Consistent configurations** in internal libraries, resource modules, and micro apps
- Improved developer experience
- Faster project setup and onboarding

## Technologies Used
- Gradle
- Kotlin DSL
- Convention Plugins
- Build tools
