# Real-time Chat System Redesign - Neon

## Context
Redesign of the real-time chat system using Jetpack Compose and Reactive Programming to improve performance and developer experience.

## Challenge/Problem
The existing chat system had performance issues and adding new chat features was slow. There was a need for faster responses and better testability.

## Your Role
Android Developer - Contributing to the development of essential features and improvements for Neon's Android banking apps, impacting more than 10 million clients.

## Actions Taken
- Redesigned the chat system using Jetpack Compose
- Implemented Reactive Programming
- Improved test coverage
- Enhanced system architecture

## Results
- **Faster responses** in the chat system
- **Greater client satisfaction**
- **50% faster addition of new chat features** due to improved tests
- Better testability and maintainability

## Technologies Used
- Jetpack Compose
- Reactive Programming
- Kotlin
- Android SDK
- Testing frameworks
