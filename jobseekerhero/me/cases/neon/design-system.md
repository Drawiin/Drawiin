# Design System Reconstruction - Neon

## Context
Reconstruction of the Design System using Jetpack Compose to reduce UI boilerplate and improve development and maintenance.

## Challenge/Problem
The existing Design System had too much boilerplate code, making UI development and maintenance slower and more error-prone.

## Your Role
Android Developer - Contributing to the development of essential features and improvements for Neon's Android banking apps, impacting more than 10 million clients.

## Actions Taken
- Reconstructed the Design System using Jetpack Compose
- Reduced boilerplate code
- Improved component reusability
- Enhanced developer experience

## Results
- **30% reduction in UI boilerplate** for better development and UI code maintenance
- Improved component reusability
- Faster UI development
- Better consistency across the app

## Technologies Used
- Jetpack Compose
- Kotlin
- Design System
- Android SDK
