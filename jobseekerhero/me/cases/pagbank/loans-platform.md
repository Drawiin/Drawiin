# Loans Platform - PagBank

## Context
Creation of the PagBank loans platform to simplify adding new products, diversify offerings, and improve client retention.

## Challenge/Problem
The existing loans system was not flexible enough to quickly add new products. There was a need to diversify offerings and improve client retention through better loan products.

## Your Role
Android Developer - Working within Invillia for PagBank, developing and improving the native digital bank app for Android, impacting more than 50 million clients.

## Actions Taken
- Created the PagBank loans platform
- Designed flexible architecture for easy product addition
- Implemented diverse loan offerings
- Focused on improving client retention

## Results
- **Simplified addition of new loan products**
- **Diversified loan offerings**
- **Improved client retention**
- Created a scalable platform for future loan products

## Technologies Used
- Kotlin
- Android
- Java
- MVVM
- Android SDK
