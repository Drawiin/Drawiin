# End-to-End Tests for Loans Platform - PagBank

## Context
Development of end-to-end test suites for the loans platform using Espresso and the Robot pattern to improve software quality and optimize testing costs.

## Challenge/Problem
The loans platform needed comprehensive testing to ensure quality, but testing costs were high. There was a need for efficient, maintainable test suites.

## Your Role
Android Developer - Working within Invillia for PagBank, developing and improving the native digital bank app for Android, impacting more than 50 million clients.

## Actions Taken
- Developed end-to-end test suites using Espresso
- Implemented the Robot pattern for test organization
- Created comprehensive test coverage for the loans platform
- Focused on test maintainability

## Results
- **Elevated software quality** through comprehensive testing
- **Optimized testing costs** for the loans platform
- Improved test maintainability
- Better confidence in releases

## Technologies Used
- Espresso
- Robot Pattern
- JUnit
- Android Testing Framework
- Kotlin
