# Exercise Platform Refactoring - Jera

## Context
Refactoring of an exercise company platform using an MVI (Model-View-Intent) approach for state management to improve testability and development speed.

## Challenge/Problem
The existing platform had complex state management that was difficult to test and maintain. Development was slow due to complexity.

## Your Role
Android Developer - Working on various client projects in health, energy, and transportation sectors, developing native Android solutions.

## Actions Taken
- Refactored the exercise platform
- Implemented MVI architecture for state management
- Improved testability
- Simplified complexity

## Results
- **Improved testability** of the platform
- **Simplified complexity**
- **50% faster development** due to better architecture
- Better code maintainability

## Technologies Used
- MVI (Model-View-Intent)
- Kotlin
- Android
- Reactive Programming
- Testing frameworks
