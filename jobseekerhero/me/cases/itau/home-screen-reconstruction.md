# Home Screen Reconstruction - Itaú

## Context
Reconstruction of the home screen of the Itaú app to improve performance and reduce server load.

## Challenge/Problem
The home screen had performance issues with loading times and was making too many requests to the server, impacting both user experience and server costs.

## Your Role
Senior Android Developer - Responsible for maintenance, development, and evolution of the Itaú app, impacting more than 100 million clients.

## Actions Taken
- Reconstructed the home screen
- Reduced loading times
- Created a new caching system
- Optimized API requests

## Results
- **Reduced loading times** significantly
- **Created a new cache system that avoids more than 45,000 requests per minute**
- Improved user experience with faster load times
- Reduced server load and costs

## Technologies Used
- Android
- Android SDK
- Kotlin
- Caching mechanisms
- MVVM
- Android Jetpack
