# Security Area Development - Itaú

## Context
Development of the security area of the Itaú app, using Angular and Kotlin for interoperability between native features and WebView.

## Challenge/Problem
Need to integrate web-based security features with native Android functionality while maintaining a seamless user experience.

## Your Role
Senior Android Developer - Responsible for maintenance, development, and evolution of the Itaú app, impacting more than 100 million clients.

## Actions Taken
- Developed the security area of the app
- Used Angular for web-based components
- Used Kotlin for native Android features
- Created interoperability between native features and WebView

## Results
- Seamless integration between web and native components
- Enhanced security features accessible through both web and native interfaces
- Improved developer experience with hybrid approach

## Technologies Used
- Angular
- Kotlin
- Android SDK
- WebView
- MVVM
